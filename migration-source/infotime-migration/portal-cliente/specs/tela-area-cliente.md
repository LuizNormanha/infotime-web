# Tela: Área do Cliente
## Origem Scriptcase: `menu_cliente_mnu` + `fluxo_cliente_lst`

### Funcionalidades Disponíveis
- Visualizar lançamentos de receita (boletos, notas fiscais)
- Solicitar segunda via de boleto
- Visualizar contratos ativos
- Alterar senha de acesso
- Avaliar o InfoLAB (`avaliacao`)

### Restrições
- Acesso apenas aos próprios dados (filtrado por `id_cliente`)
- Sem acesso a dados de outros clientes
- Interface simplificada (sem menus administrativos)
