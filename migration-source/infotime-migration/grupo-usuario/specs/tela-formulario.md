# Tela: Formulário de Grupo de Usuário
## Título real: "Inclusão Grupo de usuário"
## Screenshot: `formulario.png`

### Seção: Identificação
| Campo | Tipo | Obrigatório |
|---|---|---|
| Descrição | text | Sim |
| Id. | readonly | — |

### Seção: Fornecedor
| Campo | Tipo | Nota |
|---|---|---|
| Fornecedor | text/select | Opcional — vincula o grupo a um fornecedor |

> Formulário simples sem abas. Após criar o grupo, configurar as permissões
> na tela de Autorizações (`GrupoUsuario_AutorizacaoAcesso_Frm`).
