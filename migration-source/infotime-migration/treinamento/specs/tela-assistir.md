# Tela: Assistir Treinamento
## Origem Scriptcase: `Treinamento_Assistir_Lst`

### Campos de Controle
| Campo | Coluna DB |
|---|---|
| Usuário | `treinamento_usuario.id_usuario` |
| Treinamento | `treinamento_usuario.id_treinamento` |
| Data Início | `data_hora_inicio` |
| Data Fim | `data_hora_fim` |
| Qtd Acessos | `qtd_acesso` |

### Vídeo
- `nome_video`: nome do arquivo de vídeo
- `largura` x `altura`: dimensões do player
- Servir via streaming (não armazenar em banco)
