# Tela: Lista de Aplicações
## Origem Scriptcase: `Aplicacao_Lst`

### Colunas
| Coluna | Campo DB |
|---|---|
| Nome | `aplicacao.nome` |
| Tipo | `aplicacao.tipo` |
| Descrição | `aplicacao.descricao` |

### Tipos de Aplicação
- `Lst`: Listagem
- `Frm`: Formulário
- `Gde`: Grade detalhe
- `Blk`: Blank
- `Mnu`: Menu
- `Ctr`: Controle
- `Cal`: Calendário
- `Gra`: Gráfico
