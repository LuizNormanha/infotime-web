# Modelo de dados — auditoria

## Origem

Arquivos `.tll` em `migration-source/infotime-migration/auditoria/database/`.

## Tabelas


### `auditoria.tll`

```sql
-- Tabela: auditoria
-- Entidade: auditoria
-- Origem: MySQL → PostgreSQL (schema liga_infotime)

-- liga_infotime.auditoria  (origem MySQL: auditoria)
CREATE TABLE auditoria (
  id_auditoria bigint GENERATED BY DEFAULT AS IDENTITY NOT NULL,
  id_tenacidade bigint DEFAULT NULL,
  id_usuario_auditoria bigint DEFAULT NULL,
  data_hora_inclusao timestamp without time zone DEFAULT NULL,
  nome_aplicacao_auditoria varchar(255) DEFAULT NULL,
  nome_tabela_auditoria varchar(255) DEFAULT NULL,
  tipo_operacao_auditoria char(1) DEFAULT NULL,
  valor_id_chave_primaria_auditoria bigint DEFAULT NULL,
  endereco_ip_auditoria varchar(50) DEFAULT NULL,
  PRIMARY KEY (id_auditoria)
);
```

### `auditoria_campo.tll`

```sql
-- Tabela: auditoria_campo
-- Entidade: auditoria
-- Origem: MySQL → PostgreSQL (schema liga_infotime)

-- liga_infotime.auditoria_campo  (origem MySQL: auditoriacampo)
CREATE TABLE auditoria_campo (
  id_auditoria_campo bigint GENERATED BY DEFAULT AS IDENTITY NOT NULL,
  id_auditoria bigint DEFAULT NULL,
  nome_campo varchar(100) DEFAULT NULL,
  valor_anterior text,
  valor_atual text,
  PRIMARY KEY (id_auditoria_campo)
);
```
